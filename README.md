# Edge-First LLM Tool Calling Framework

A progressive enhancement framework for running LLM inference with tool calling capabilities directly in the browser.

## Packages

- **@edge-llm/core** - Core runtime for edge-first LLM tool calling
- **@edge-llm/react** - React hooks and components

## Development

```bash
# Install dependencies
npm install

# Build all packages
npm run build

# Run tests
npm run test

# Start example app
npm run dev
```
